1️⃣ Toujours récupérer les changements avant de travailler
--> git pull

-------------------------------------------------------------------

2️⃣ Travailler normalement
Tu modifies tes fichiers :

.py
notebooks
configs
etc.

--------------------------------------------------------------------

3️⃣ Ajouter tes modifications

-->  git add .

--------------------------------------------------------------------

4️⃣ Faire un commit

-->  git commit -m "ajout preprocessing texte"
Le commit doit expliquer ce que tu as fait.

--------------------------------------------------------------------

5️⃣ Envoyer sur GitHub

-->  git push

--------------------------------------------------------------------

⚠️ Cas très courant

Si quelqu'un a push pendant que tu travaillais, git push peut refuser.
Dans ce cas :

-->  git pull

puis :

-->  git push
